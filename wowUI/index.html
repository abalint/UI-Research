All original CSS — no Blizzard texture files used. Metal & paper grain are generated procedurally with an SVG feDiffuseLighting filter (real surface height, not a flat gradient). Every element carries its own carved metal bezel; depth uses directional bevels — raised (light top / dark base) for buttons, portal & rivets, sunken (dark top / light lip) for the wells, chips & list. See RESEARCH.md for the construction notes.
